# arnaldo antunes
⚠️ esta página foi desenvolvida para atender à proposta abaixo:

> crie uma página html para apresentar o seu cantor(a) ou grupo musical favorito.
> 
> estruture o seu trabalho como um projeto (arquivos separados para html, css e javascript).
> 
> no arquivo javascript, utilize o método ```document.querySelector()``` para alterar os atributos dos elementos como você quiser.
>  
> use a criatividade.
> 
> a página deverá conter as seguintes informações:
>
> * informações sobre você (nome, idade, estado, foto);
> informações sobre seu cantor(a) ou grupo musical favorito (nome, gênero musical, links para consulta, imagens, lista de suas músicas preferidas);
